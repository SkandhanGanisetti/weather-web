#  Weather App

A weather forecast web app that displays real-time weather info for any city. It supports:

-  Voice input
-  Auto-detect location
-  Background changes based on weather
-  Stylish glassmorphism UI

##  Tech Stack

- HTML, CSS, JavaScript
- OpenWeatherMap API

##  How to Run

1. Open `index.html` in a browser (use Live Server or `python -m http.server` for full features)
2. Enter a city name or use the mic button
3. See the weather details and animated background

##  API Used
[OpenWeatherMap](https://openweathermap.org/api)
